# O'rnatish va foydalanish ko'rsatmalari

## O'rnatish
1. Sizga Git mavjud bo'lishi kerak. Devizdan foydalanib, quyidagi buyruqni bajaring:
```bash
sudo apt-get install git
```

2. Repozitoriyani klonlash uchun terminalda quyidagi buyruqni bajaring:
```bash
git clone https://github.com/saburovvv/pay.git
```

3. Loyihani kutubxonalar bilan uzluksiz ishlash uchun `npm` yoki `yarn` orqali talab qilinadigan paketlarni o'rnating. 
```bash
npm install
```
// yoki
```bash
yarn install
```

## Foydalanish
- Dastur ishlatish uchun quyidagi buyruqni bajaring:
```bash
npm start
```

Oyoq bilan bog'liq dasturdan foydalanish uchun ekranda ko'rsatilgan ko'rsatmalarga amal qiling.
